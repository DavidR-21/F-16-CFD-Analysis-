F-16 Aerodynamic Analysis

This project analyzes the aerodynamic performance of an F-16 aircraft using ANSYS.

## Overview
A 3D model of the aircraft was used to simulate airflow and evaluate lift and drag characteristics under varying conditions.

## Tools Used
- ANSYS (CFD)
- CAD Modeling

## Key Features
- Computation of lift and drag coefficients
- Analysis of airflow behavior around the aircraft
- Evaluation of aerodynamic efficiency

## Results
The simulation provided insight into how flight conditions affect lift and drag, with results compared to theoretical expectations.
